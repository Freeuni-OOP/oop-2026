## 1. Math
бғЁбғ”бғҘбғӣбғ”бғңбғҳбғ— `Math` бғҷбғҡбғҗбғЎбғҳ бғ бғқбғӣбғ”бғҡбғЎбғҗбғӘ бғ”бғҘбғңбғ”бғ‘бғҗ бғқбғ бғҳ бғӨбғЈбғңбғҘбғӘбғҳбғҗ `add` бғ“бғҗ `abs`. бғ’бғҗбғӣбғқбғҳбғ§бғ”бғңбғ”бғ— TDD бғӣбғҳбғ“бғ’бғқбғӣбғҗ.


## 2. MyStack
бғЁбғ”бғҘбғӣбғ”бғңбғҳбғ— бғҜбғ”бғңбғ”бғ бғҳбғҘ `MyStack` бғҷбғҡбғҗбғЎбғҳ TDD бғӣбғҳбғ“бғ’бғқбғӣбғҳбғ—, бғ бғқбғӣбғ”бғҡбғЎбғҗбғӘ бғ”бғҘбғңбғ”бғ‘бғҗ 4 бғӣбғ”бғ—бғқбғ“бғҳ:
1. add(T element) - бғҗбғӣбғҗбғўбғ”бғ‘бғЎ бғҗбғ®бғҗбғҡ бғ”бғҡбғ”бғӣбғ”бғңбғўбғЎ
2. T pop() - бғЁбғҡбғҳбғЎ бғ“бғҗ бғҗбғ‘бғ бғЈбғңбғ”бғ‘бғЎ бғ‘бғқбғҡбғқ бғ”бғҡбғ”бғӣбғ”бғңбғўбғЎ
3. addAll(Collection<T> elements) - бғҗбғӣбғҗбғўбғ”бғ‘бғЎ бғ§бғ•бғ”бғҡбғҗ бғ”бғҡбғ”бғӣбғ”бғңбғўбғЎ бғ’бғҗбғ“бғҗбғӘбғ”бғӣбғЈбғҡбғҳ бғҷбғқбғҡбғ”бғҘбғӘбғҳбғҳбғ“бғҗбғң
4. size() - бғҗбғ‘бғ бғЈбғңбғ”бғ‘бғЎ бғЎбғўбғ”бғҷбғЁбғҳ бғ”бғҡбғ”бғӣбғ”бғңбғўбғ”бғ‘бғҳбғЎ бғ бғҗбғқбғ“бғ”бғңбғқбғ‘бғҗбғЎ
